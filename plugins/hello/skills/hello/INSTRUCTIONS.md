# Hello Skill

A simple greeting skill to demonstrate the plugin structure.

## Usage

Invoke with `/hello` or `/hello [name]`

## Behavior

When invoked:

1. If a name is provided, respond with "Hello, {name}!"
2. If no name is provided, respond with "Hello, world!"

Keep the response brief and friendly.

## Examples

- `/hello` -> "Hello, world!"
- `/hello Alice` -> "Hello, Alice!"
- `/hello Claude` -> "Hello, Claude!"
